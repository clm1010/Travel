# 旅游

> 一个Vue.js项目

## 构建设置

``` 使用cmd或bash命令
# 安装依赖关系
npm install

# 在localhost：8080进行热重新加载
npm run dev

# 打包压缩建立生产
npm run build

# 建立生产和查看捆绑分析器报告
npm run build --report
```

有关如何工作的详细说明，请查看
[guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
